# bootsoundplayer


# Whats this?

## this is a minimal program that will play anything you put in there, but it's focused on system sounds

# Using

## Simply put the audio file in audio folder, and run the run.sh bash script

## for the inicialization, configure it in your DE or WM, WARNING: the folder of the project needs to be on the your home folder, and the folder name needs to be "bootsound", fulfilling these requirements, you should be able to play the sound from anywhere by running the run.sh script

# Building

```
make
```
